# ICMF Framework

ICMF (Insider Code Manipulation Framework) is a cybersecurity framework designed to detect insider-driven manipulation in enterprise software systems.

The framework focuses on identifying manipulation techniques in:

- Financial logic manipulation
- Authorization bypass mechanisms
- Audit log tampering
- Data integrity manipulation
- Cross-system abuse

ICMF defines a taxonomy of insider code manipulation techniques and provides a detection model for security tools and auditors.
