# ICMF Technique Registry

ICMF defines a set of insider code manipulation techniques used to identify insider-driven manipulation in enterprise software systems.

| Technique ID | Technique Name | Category | Risk |
|---------------|---------------|---------|------|
| ICMF-T001 | Shadow Mode Backdoor | Authorization Manipulation | Critical |
| ICMF-T002 | Financial Rounding Manipulation | Financial Manipulation | Critical |
| ICMF-T003 | Audit Log Tampering | Audit Manipulation | Critical |
| ICMF-T004 | Preferred User Bypass | Authorization Manipulation | High |
| ICMF-T005 | Time Bomb Logic | Cross-System Manipulation | High |
