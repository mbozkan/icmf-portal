# ICMF Portal

Public framework portal for ICMF (Insider Code Manipulation Framework).

## Purpose
ICMF is a cybersecurity framework focused on detecting insider code manipulation risks in enterprise systems such as ERP, financial software, and large backend platforms.

## Website
https://icmf.dev

## Status
Initial public portal repository. Content, taxonomy, techniques, whitepaper integration, and framework governance documents are being prepared.
